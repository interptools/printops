Setup guide
1
Create a free Supabase account
Go to supabase.com → sign up free → create a new project. Choose any region closest to the Philippines (e.g. Singapore).
2
Create the database tables
In your Supabase dashboard, go to SQL Editor and run this query:

create table po_orders (id serial primary key, data jsonb);
create table po_customers (id serial primary key, data jsonb);
create table po_invoices (id serial primary key, data jsonb);
3
Get your project URL and anon key
Go to Project Settings → API. Copy the Project URL and the anon public key. Paste them in the fields above.
4
Test and push your data
Click Test connection to verify. Then click Push local data to cloud to upload your existing orders, customers, and invoices. From any device, click Pull data from cloud to sync.
